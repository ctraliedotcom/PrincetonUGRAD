Pythagorean identities:

1) sin2(x) + cos2(x) = 1

PROOF:

This should be fairly obvious based on the reference triangle of hypotenuse one (since the trig functions are defined based on the unit circle). Referring to the above picture:

cos(A) = x                    sin(A) = y

sin(A)2 + cos(A)2 = y2 + x2 = 1  (because of the Pythagorean theorem)

 

2) tan2(x) + 1 = sec2(x)

PROOF:

Start with the first identity: sin2(x) + cos2(x) = 1, and divide both sides by cos2(x)

 

tan2(x) + 1 = sec2(x)

 

3) cot2(x) + 1 = csc2(x)

PROOF:

Once again, start with the identity sin2(x) + cos2(x) = 1, and this time divide by sin2(x)

1 + cot2(x) = csc2(x)
