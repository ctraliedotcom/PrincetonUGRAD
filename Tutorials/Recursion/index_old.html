Intro to Number Systems
Chris Tralie
ACSL Math
This tutorial introduces some fundamental concepts, such as binary, that perpetually resurface all throughout the ACSL contest. This is one of the most important precursor tutorials for later lessons as well, so be sure to look at it if you are new to any of the material.
Recursion
Chris Tralie
ACSL Math & Enrichment
Recursion is one of the topics on the first ACSL exam, and it also just so happens to be one of my favorite mathematical concepts. To prepare for the ACSL exam, you only really need to look at the first subsection of this tutorial, which is really quite short. However, I don't feel that the ACSL exam does recursion justice, so also created many subsections of this tutorial dealing with practical applications of recursion. I am very proud of these sections, and I think that they help to have a deeper understanding of the concepts, so I would highly recommend looking them over (I also created some cool applets).

Perl Programming
Chris Tralie
Enrichment Programming
Perl is a great, easy language to use for the ACSL contest. I have often been able to condense ACSL programs to less than 30 lines with this language! Instead of writing a full-blown tutorial on Perl (there are plenty of programming tutorials online), I made this tutorial a few years ago as a sort of "cram sheet" to accompany an introductory lesson on Perl. It's a very succint Microsoft Word document with everything you need to get started. To install Perl under a Windows environment, either download Active Perl or follow my instructions for installing Cygwin with the Perl package.
Boolean Algebra
Chris Tralie
ACSL Math
Boolean algebra is one of the main focusses of the second ACSL contest. I personally enjoy it very much, as it is all very logical and elegant, and is also important for explaining the behavior of digital circuits (as you will see later).
