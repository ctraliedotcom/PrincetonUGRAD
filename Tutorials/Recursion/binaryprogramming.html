Bits and bytes

As you now know, everything in a computer is stored as a series of 1s and 0s, and each binary digit is known as a bit for short (it takes too long to say "binary digit"). Unfortunately, there is only a finite amount of RAM and hard drive space, so we can't just assume that we have all of the bits that we need. To keep track of all of the bits that we do have, we can use the SI prefixes. In other words, a "megabit" is 106 (a million) bits, and a "gigabit" is 109 (a billion) bits. We can also describe quantities in terms of a byte, which is eight bits (computer programmers are witty sometimes). So if you have 256 megabytes of RAM, you actually have 2.048 x 109 little bit switches working to store everything you have loaded. With this system, each byte can be expressed as two hexadecimal digits.

In C++ and Java, some common data types and their size in bytes are as follows:
char1 byte
short1 byte
int2 bytes
float4 bytes
double4 bytes
